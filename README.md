# 今日のToDo

ブラウザだけで動く、シンプルなToDoアプリです。サーバーや外部ライブラリは必要ありません。

## 主な機能

- タスクの追加、編集、完了、削除、削除の取り消し
- 期限、優先度、カテゴリー
- 期限切れの強調
- 検索、状態フィルター、並べ替え
- 完了履歴
- JSON形式のバックアップ書き出し、読み込み
- ライトモード、ダークモード
- スマートフォン表示
- ブラウザ内への自動保存

## 使い方

`index.html`をブラウザで開きます。入力した内容は、そのブラウザの`localStorage`に保存されます。

別のブラウザや端末へ移す場合は、画面下部の「書き出す」でバックアップを保存し、「読み込む」で復元してください。

## テスト

Node.jsとMicrosoft EdgeがあるWindows環境で実行できます。

```powershell
node tests/verify-app.cjs
```

Edgeの場所が異なる場合は、`EDGE_PATH`環境変数で指定します。

## GitHub Pagesで公開

1. このフォルダをGitリポジトリとして初期化します。
2. GitHubで空のリポジトリを作成し、`main`ブランチをプッシュします。
3. GitHubのリポジトリで `Settings` → `Pages` を開きます。
4. `Deploy from a branch`を選び、`main`と`/(root)`を指定して保存します。

公開後のURLは通常、`https://ユーザー名.github.io/リポジトリ名/`です。
